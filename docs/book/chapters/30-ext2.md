# Chapter 30 — The ext2 Filesystem (Read-Only)

> Part VI — Storage & Filesystem · Status: 🚧 outline

> **Source files:** `fs/ext2.c`, `fs/ext2.h`, `scripts/mkinitrd.sh`

## What this chapter covers

The final layer: reading a real on-disk filesystem. SeedOS implements a read-only
**ext2** reader — today over the RAM-disk module, and (after Chapter 29) over a
real virtio block device. This chapter covers the ext2 on-disk format and the
reader that walks it.

## Outline

1. **What a filesystem is** — turning a flat array of blocks into files and
   directories.
2. **ext2 on disk** — the superblock, block groups, inodes, and directory
   entries.
3. **Intuition** — an inode is a file's metadata record; directories are just
   files listing name → inode.
4. **In SeedOS** — `ext2_init`, `ext2_lookup`, `ext2_read_inode`,
   `ext2_read_file`; mounting the initrd at boot.
